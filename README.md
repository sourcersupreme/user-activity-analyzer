# User Activity Monitor (GUI Based)

## ⚠️ Disclaimer
This project is developed for educational purposes only.
It demonstrates how keystroke logging works to help understand
security risks and detection mechanisms.

## Features
- GUI-based interface
- Real-time keystroke monitoring
- Active window tracking
- Structured logging format

## Log Format
[Time] | [Application] | [Key]
